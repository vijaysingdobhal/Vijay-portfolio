## üöÄ Getting Started

This is a static website, so you can run it locally without any complex setup.

Prerequisites:
-   A modern web browser (like Chrome, Firefox, or Edge).
-   A code editor (like VS Code) if you wish to make changes.

Installation:
1.  Clone the repository:
        git clone https://github.com/vijaysingdobhal/Portfolio.git
    
2.  Navigate to the project directory:
        cd Portfolio
    
3.  Open the `index.html` file:
    Simply double-click the index.html file in the project folder to open it in your default web browser.

## ‚úçÔ∏è How to Customize for Your Own Use

This portfolio was designed to be easily personalized.

1.  Update Personal Information (`index.html`):
    -   Change the name, titles, and descriptions in the Hero (<header>), Summary (#summary), and Contact (#contact) sections.
    -   Update the email address in the "Say Hello" button link.
    -   Replace the social media links in the footer (<footer>).

2.  Modify Skills (`index.html`):
    -   In the "My Tech Stack" section (#skills), change the icons and text for each skill-card. You can find more icons at [Font Awesome](https://fontawesome.com/search).

3.  Update Projects (`index.html`):
    -   For each project-card in the "Featured Projects" section (#projects), replace the title, description, tags, and project link.
    -   Replace the placeholder project images (assets/projects/*.jpg) with screenshots of your own work.

4.  Replace Resume:
    -   Add your own resume PDF to the assets/ folder.
    -   Update the link in the navbar to point to your new file: <a class="btn btn-resume" href="assets/Your_Resume_Name.pdf" ...>.

5.  Change Images:
    -   Replace assets/profile.jpg with your own profile picture.
